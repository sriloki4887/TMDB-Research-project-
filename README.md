# TMDB-Research-project-

My project is "Predicting Movie Box Office Revenue Using Machine Learning." The aim was to determine whether movie features such as budget, runtime, genre, popularity, vote average, and vote count can predict box office revenue.

I used movie data from TMDB, performed data cleaning and exploratory data analysis, and then trained four machine learning models: Linear Regression, Random Forest, Gradient Boosting, and MLP.

To improve performance, I applied hyperparameter tuning using RandomizedSearchCV with cross-validation. The models were evaluated using MAE, RMSE, and R² Score.

The results showed that Gradient Boosting was the best-performing model, achieving the highest R² score and lowest prediction error. This demonstrates that machine learning can effectively predict movie revenue using movie-related features.

In the future, additional features such as marketing budget, actor popularity, and social media trends could further improve prediction accuracy.
